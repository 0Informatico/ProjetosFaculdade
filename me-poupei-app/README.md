# Me Poupei - Assistente Financeiro com IA

App de controle financeiro pessoal com assistente de inteligência artificial integrado.

## Funcionalidades

- Chat por voz com IA para consultas financeiras
- Chat de texto com IA
- Registro de gastos e entradas
- Análise visual de gastos com gráficos
- Interface responsiva (mobile e desktop)

## Stack

- React + Vite
- Tailwind CSS + shadcn/ui
- Lucide Icons

## Como rodar

```bash
npm install
npm run dev
```
