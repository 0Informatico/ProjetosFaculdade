# Exercícios Java - Programação Orientada a Objetos

Coleção de exercícios acadêmicos de POO em Java.

## Conteúdo

- **basico/** - Exercícios fundamentais de Java (14 programas)
- **heranca-polimorfismo/** - Herança, polimorfismo e classes abstratas (Conta, Animal, Pessoa, Ingresso, Imóvel, Funcionário)
- **composicao/** - Composição e associação (Livro, Animal, Texto, Venda, Lâmpada, Contador, Data)
- **threads/** - Programação concorrente com Synchronized
- **windowbuilder/** - Interface gráfica com WindowBuilder (Calculadora)
