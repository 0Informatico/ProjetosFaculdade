# Scripts de Automação Python

Coleção de scripts de automação para diversas tarefas.

## Scripts

- **beatsaver_downloader.py** - Download automático de playlists do BeatSaver usando Selenium
- **auto_clicker.py** - Auto-clicker com reconhecimento de imagem (PyAutoGUI)
- **link_opener.py** - Abre links de um arquivo de texto sequencialmente
- **text_formatter.py** - Formatador de texto copiado (remove quebras de linha indesejadas)
- **plant_tracker.py** - Tracker de plantações com informações por estação
- **blender_script.py** - Script de automação para Blender 3D

## Dependências

```bash
pip install selenium pyautogui keyboard clipboard
```
