# Estruturas de Dados em Java

Implementações de estruturas de dados clássicas.

## Conteúdo

- **arvore-binaria/** - Árvore binária de busca com inserção e travessia
- **lista-encadeada-simples/** - Lista encadeada simples com operações CRUD
- **lista-encadeada-dupla/** - Lista duplamente encadeada
- **lista-circular/** - Lista circular
- **fila-fifo/** - Fila FIFO (First In, First Out)
- **fila-prioridade/** - Fila com prioridades
- **cache-lru/** - Algoritmo de substituição LRU (Least Recently Used)
- **cache-opt/** - Algoritmo de substituição OPT (Ótimo)
