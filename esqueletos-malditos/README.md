# Esqueletos Malditos - Gerador de Ataques RPG

Ferramenta com interface gráfica para gerar ataques e danos de esqueletos em sessões de RPG de mesa.

## Funcionalidades

- Rolagem de dados de ataque (d20 + bônus)
- Cálculo de dano (d6 + bônus)
- Tipos de ataque: espada e arco
- Interface gráfica com Tkinter

## Como rodar

```bash
python main.py
```
