# Buscador de Documentos

Motor de busca simples com índice invertido para busca em documentos textuais.

## Funcionalidades

- Indexação de documentos com índice invertido
- Busca por múltiplos termos (intersecção)
- Filtragem de stopwords
- Índice refinado com remoção de palavras irrelevantes

## Como rodar

```bash
python buscador.py
```
