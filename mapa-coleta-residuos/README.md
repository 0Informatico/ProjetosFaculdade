# Mapa de Coleta de Resíduos - São Bento do Sul/SC

Mapa interativo com pontos de coleta de lixo reciclável e eletrônico usando GIS.

## Funcionalidades

- Mapa interativo em HTML gerado com Folium
- Pontos de coleta categorizados (reciclável, eletrônico, áreas verdes)
- Geolocalização dos ecopontos SAMAE

## Stack

- Python
- Folium (mapas interativos)
- GeoJSON

## Como rodar

```bash
pip install folium
python mapa.py
```

O mapa será gerado como arquivo HTML.
