# Projeto Inclusive

Plataforma web para busca de profissionais de inclusão escolar, escolas e recursos educacionais.

## Funcionalidades

- Busca de profissionais especializados
- Listagem de escolas inclusivas
- Área de recursos educacionais
- Painel administrativo
- Interface responsiva

## Stack

- React + Vite + React Router
- Tailwind CSS + shadcn/ui
- Lucide Icons

## Como rodar

```bash
npm install
npm run dev
```
