# Trabalho Final - Padrão Mediator em Java

Aplicação interativa usando o padrão de projeto Mediator com interface gráfica Swing.

## Funcionalidades

- Cronômetro integrado
- Sistema de jogadores com tela dedicada
- Comunicação entre componentes via Mediator
- Interface gráfica com botões de ação

## Design Pattern

Implementa o padrão **Mediator** para desacoplar a comunicação entre componentes de UI.
