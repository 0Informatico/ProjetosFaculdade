# Buckshot Roulette Tracker

Ferramenta visual para acompanhar balas (vivas e de festim) no jogo Buckshot Roulette.

## Funcionalidades

- Contagem de balas vivas e de festim
- Inversão de contagem
- Interface visual com imagens das balas

## Stack

- React + Vite

## Como rodar

```bash
npm install
npm run dev
```
